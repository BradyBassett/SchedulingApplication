## How to run app
    First create your own instance of this databse, following this ERD diagram
<img src="/src/com/C195/Resources/ERD.png/" width="100%" alt="see src for image">
    
    Once that has been done, remove the .example portion from db.example.properties and insert all of 
    your database information in order for a connection to your database to be possible. After this has
    completed, you can locate the main file and run the application. Alternatively you may create a build
    and run the application from your newly compiled build.
